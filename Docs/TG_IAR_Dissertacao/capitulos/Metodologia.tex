\chapter{Metodologia}\label{cap:metod}

O sistema de controle da antena será baseado em um pedestal convencional de antena,
temos o controle de dois parâmetro, sendo estes: azimute e elevação.
Para efetuar o controle desses parâmetros, serão necessários dois motores que atuarão modificando
os ângulos mencionados anteriormente. O controle do azimute será feito por 360 graus, enquanto o
controle de elevação atuará somente em 90 graus. Desta maneira, o ângulo de visada total do mecanismo
será dado por uma meia-esfera superior. Essa geometria é considerada mais que suficiente para a detecção e o rastreio de qualquer satélite necessário pela estação de solo.

Existem dois canais no encoder defasados $90^{\circ}$ entre eles (vide figura \ref{fig:canais_encoder}). A defasagem faz com que, ao comparar os dois sinais, existam quatro estados distintos para o sistema.
Os estados intermediários gerados permitem que o encoder tenha uma resolução virtual de 360 pulsos (segundo tabela \ref{tab:num_estados_encoder}). Além disso, ao verificar os estados do encoder, existe uma ordem pela qual o sistema irá passar.

Os $360^{\circ}$ por volta resultam em uma resolução de $1^{\circ}$, que é baixa para a aplicação desejada. Uma forma simples de melhorar a resolução é inserir um acoplador
com relação de transmissão dada por $i=\frac{\omega_ams}{\omega_obs}$, onde $\omega_ams$ é a velocidade angular do sistema amostrado, e $\omega_obs$ é a velocidade angular observada no encoder.
Desse modo, ao obter uma relação de transmissão $i>1$, a nova resolução do sistema será dada por $\frac{1}{i}$ graus.

\section{Motores}
Para os motores do projeto, serão utilizados dois motores Pittman com caixa de redução reversível de 12 VDC, modelo GM9413-3 da marca Pittman.
Esse motor possui uma razão de redução de $65,5:1$. Para motivos de simplificação, será observada somente a rotação do motor após a caixa de redução.
Os testes realizados foram utilizando um programa especial que envia uma variação de 1024 estados (10 bits de resolução) de tempo de trabalho (duty cycle, DC) a uma frequência de 5 kHz para uma ponte H, na qual o motor é conectado (vide figura \ref{fig:setup_motor}). Ambos motores foram girados tanto para sentido horário (indicado por "HOR") e anti-horário ("ANTI").
O programa faz duas medidas, sendo uma para o duty cycle crescente e uma para decrescente. Esse teste é feito para determinar alguns parâmetros do motor, como linearidade e porcentagem de duty cycle para partida ou parada.
A união entre o motor e o encoder foi feita utilizando um acoplador flexível de 6 mm x 8 mm. Este acoplador serve para unir os dois eixos de tamanhos diferentes (8 mm do motor e 6 mm do eixo do encoder), além de permitir a passagem de potência mesmo com desalinhamento angular dos eixos.
Como o encoder e o eixo possuem pouco momento de inércia quando comparado ao motor, é plausível inferir que o motor está funcionando em vazio.

\subsection{Cálculo para a Elevação}\label{subsec:calc_elev}
Para o mecanismo de elevação, será proposto um novo sistema para aumentar a resolução utilizando um fuso de rosca trapezoidal. A ideia por trás do mecanismo é criar um triângulo utilizando como lados o poste do pedestal, o braço de alavanca da antena, e o fuso. A vantagem desse sistema é de conseguir uma grande resolução utilizando o passo do fuso como rotação.
A equação que descreve o ângulo em função da posição do fuso é a lei dos cossenos, definida por $c^2 = a^2 + b^2 - 2ab\cos\theta$ que, ao manipular, é possível obter $\theta = \arccos\left(\frac{c^2 - (a^2 + b^2)}{-2ab}\right)$. 

A equação correlaciona o ângulo $\theta$ em uma função $\theta(c)$. 
Considerando então $a = \text{tamanho do poste}$, $b = \text{tamanho do braço da antena}$, 
e $c = \text{tamanho do fuso}$, é possível calcular o parâmetro de $\theta$. Assim, segundo a seção \ref{sec:desc_az_el}, é possível encontrar uma variação em $c$ correspondente a um ângulo compreendido entre $0^\circ\le\theta\le90^\circ$.
