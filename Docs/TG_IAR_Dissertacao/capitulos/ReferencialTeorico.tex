Para iniciar a modelagem do sistema, é necessário descrever o mecanismo.
Segundo \citeauthor{Ford2008}, o sistema pode ser modelado utilizando como base um (** A definição seria essa, mas é necessário confirmar e encontrar um modelo mostrando a dinâmica do sistema) manipulador esférico de dois graus de liberdade.
O ponto do modelo considerado como fixo será o da base, como visto no modelo (** inserir modelo).
É possível descrever o mecanismo do pedestal a partir de dois movimentos independentes, sendo um de azimute e um de elevação.
Para descrever uma semi-esfera, pode-se utilizar uma rotação de $0$ até $2\pi$ em azimute e $0$ até $\frac{\pi}{2} $ na elevação.
